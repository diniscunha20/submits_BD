## ​7.2 
 
### *a)*

```
Está na 1 forma normal porque os atributos são atómicos (simples e indivisíveis) e não suporta relações dentro de relações (Nested Relation)
It is possible to <u>underline</u> -> {A, B,.. }
-> R1 (_A_, B, C)
-> R2 (B,F)
```

```

```
